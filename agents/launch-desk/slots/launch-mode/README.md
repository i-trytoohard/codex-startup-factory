# Launch Mode Slot

This slot holds configurable launch-planning and outbound-execution strategies for the `launch-desk` agent.

Each config selects:

- a prompt source
- launch surfaces
- audience tracks
- outreach intensity
- an output shape
- expected asset outputs
- success signals

Current configs:

- `codex.json`: founder-led launch mode for Product Hunt, Hacker News, social, PR, and warm intros
