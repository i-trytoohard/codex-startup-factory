# AI Debates Social App

A mobile-first demo for an `X for debates` product. The app presents a social feed of provocative debate matchups between notable living and historical figures, then opens a detail view with a static transcript, related conversation, and community comments.

## What is included

- A user-facing trending feed with editorial debate cards
- A curated `Arena` tab
- A `Rankings` tab with debate performance stats
- A `Settings` tab for lightweight product preferences
- Debate detail pages with:
  - transcript
  - related X, Reddit, and news references
  - likes
  - community discussion

## Run locally

```bash
npm install
npm run dev
```

The Vite dev server will start on the default local port.

## Build

```bash
npm run build
```

## Stack

- React
- TypeScript
- Vite

## Notes

This example is intentionally self-contained inside the repository and does not participate in the top-level pnpm workspace packages.
